Elementos de Citação do HTML

Nesta aula, vamos abordar os elementos HTML <blockquote>, <q>, <abbr>, <address>, <cite> e <bdo>

Exemplo:

<blockquote>

O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo navegador com recuo. A URL para a fonte da citação pode ser dada usando o atributo cite.

Aqui está uma citação do site da WWF:

Por 60 anos, a WWF tem trabalhado para ajudar pessoas e a natureza a prosperarem. Como a principal organização de conservação do mundo, a WWF atua em quase 100 países. Em todos os níveis, colaboramos com pessoas ao redor do mundo para desenvolver e entregar soluções inovadoras que protegem comunidades, a vida selvagem e os lugares onde vivem.

<q>

Também podemos fazer pequenas citações utilizando a tag <q>

Exemplo:

WWF's goal is to: Build a future where people live in harmony with nature.

Os navegadores geralmente insirem as aspas nas extremidades da citação quando utilizamos este formato.

<abbr>

A tag HTML <abbr> define uma abreviação ou um acrônimo, como “HTML”, “CSS”, “Sr.”, “Dr.”, “ASAP”, “ATM”. Marcar abreviações pode fornecer informações úteis para navegadores, sistemas de tradução e mecanismos de busca. Dica: Use o atributo global title para mostrar a descrição da abreviação/acrônimo quando você passar o mouse sobre o elemento.

Exemplo:

A WHO foi fundada em 1948.

<adress>

A tag HTML <address> define as informações de contato do autor/proprietário de um documento ou artigo. As informações de contato podem ser um endereço de email, URL, endereço físico, número de telefone, identificador de mídia social, etc. O texto no elemento <address> geralmente é renderizado em itálico, e os navegadores sempre adicionam uma quebra de linha antes e depois do elemento <address>.

Exemplo:

Escrito por Max Roslaniec.
Visite nossa sede:
calmanegativa.com.br
Alameda dos Anjos, 123
Porto Alegre-RS

<cite>

A tag HTML <cite> define o título de uma obra criativa (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.). Nota: O nome de uma pessoa não é o título de uma obra. O texto no elemento <cite> geralmente é renderizado em itálico.

Exemplo

O Grito por Edvard Munch. Pintado em 1893.

<bdo>

BDO significa Substituição Bidirecional. A tag HTML <bdo> é usada para substituir a direção atual do texto:

Exemplo

Este texto será escrito da direita para a esquerda
